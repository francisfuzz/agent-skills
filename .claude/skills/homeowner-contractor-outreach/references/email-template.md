# Quote Request Email Template

Use this template as a starting point. Personalize the greeting and any
bracketed fields before sending. Keep it under 150 words — contractors are busy.

---

**Subject**: Quote Request — {Trade Type} Service at {City, State}

Hi {Contractor First Name or "there"},

I'm a homeowner in {City, State} and I'm looking for a licensed {trade type}
to help with the following:

{2–3 sentence description of the problem in plain language. Be specific:
what's broken, when it started, what you've already tried if anything.}

{If urgent: "This is somewhat urgent — I'd love to get this addressed within
{timeframe} if possible."}

I'm available for a site visit or call during: {preferred_contact_window}.

Could you provide a rough estimate, or let me know what additional information
you'd need to do so?

Thanks so much,
{Homeowner First Name}
{Phone number — optional, include if homeowner wants callbacks}

---

## Variations

**For emergency / same-day**:
Change the opening to:
> "I have an urgent {trade type} issue at my home in {City} and am looking
> for someone who can come out today or tomorrow."

**For large projects (remodel, roof, etc.)**:
Add after the problem description:
> "I'm getting quotes from a few contractors before making a decision.
> I'm hoping to start the project by {rough timeframe}."

**Follow-up (no reply after 3 days)**:
> Subject: Following up — Quote Request for {Trade Type}
>
> Hi {Name}, just wanted to follow up on my message from {date}. I'm still
> looking for a {trade type} in {City} — let me know if you're available
> and interested. Thanks!
